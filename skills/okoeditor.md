## Edycja systemu

Use `execute_oko_edits` to perform all required changes in the OKO system. This includes changing incident classifications, updating tasks, and adding new incident reports.

## Finalizacja

Use `finalize_oko` to run the "done" action and retrieve the flag. Call this only after all edits are complete.
